Features

What's in the build

The whole feature set, plainly listed — no upsells gating the basics.

◔

Tick-mark dial

A pure-RN circular dial — no game-engine dependency. Each tick fills as the seconds pass.

✦

Local alerts

An end-of-session notification is scheduled at start, so it fires even if you background the app or the screen locks.

↻

Auto-cycle

Focus → break → focus, automatically. Custom focus and break lengths in Settings.

☼

Today at a glance

A small stat strip below the dial: sessions completed today, minutes focused, sessions all-time.

◯

Keeps the screen awake

While a session is running, the display stays on so the dial is always visible.

⊘

No history sold

Only the last 200 focus sessions are kept on-device to compute today. They are never uploaded.

How to play

From cold-start to good-at-it

A few short steps. The app gets out of your way after that.

01

Tap Start

The dial begins winding down twenty-five minutes by default. A haptic confirms.

02

Work

Put the phone down. When the dial completes, the notification fires and the mode flips to a five-minute break.

03

Take five

The break dial counts down. When it ends, the mode flips back to focus and you can start the next round.

04

Adjust if you like

Open Settings to change focus and break lengths (1–120 / 1–60 minutes).
